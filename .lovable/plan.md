
## Holbi-Inspired B2B Ecommerce Agency Site

A modern, dark-themed marketing site inspired by Holbi Group — recreated as a fresh brand with refined typography, layout, and visuals. Built as a multi-page React site.

### Brand direction
- **Vibe:** Dark, confident, enterprise. Cleaner and more contemporary than the original.
- **Palette:** Deep navy/near-black background, off-white text, a single vibrant accent (electric teal/cyan) for CTAs and highlights.
- **Type:** Large display headings (tight tracking) + clean sans body. Generous whitespace.
- **Motion:** Subtle fade/slide-in on scroll, hover lifts on cards, smooth section transitions.
- **Placeholder brand name:** "Nordstack" (easy to swap later).

### Pages

**1. Home**
- Sticky transparent nav (logo, links, Contact CTA)
- Hero: full-bleed dark image + headline "Scalable B2B Ecommerce Platforms", subline, primary CTA, two big stats (25+ Years / 5000+ Projects)
- Platform highlight section (osCommerce-style) with phone mockup and 3 mini-stats
- Featured clients/case study carousel (logo + screenshot cards)
- Services overview grid (4 cards linking to subpages)
- Process / "Why us" section (3–4 steps with icons)
- Testimonial or large quote
- Contact CTA strip + footer

**2. E-commerce Development**
Hero, capabilities grid, tech stack badges, sample work, CTA.

**3. B2B Enterprise**
Hero, feature list (multi-store, pricing tiers, API, ERP integrations), comparison table, CTA.

**4. Hosting**
Hero, plan cards (Starter / Pro / Enterprise) with features, uptime/SLA badges, FAQ, CTA.

**5. Datalink**
Hero explaining the data sync product, diagram of integrations, key benefits, CTA.

**6. Contact**
Two-column: contact info + form (name, company, email, message). Form is **visual-only with a success toast** for now — easy to wire to a backend later if you want.

### Shared components
Top nav with mobile menu, footer with sitemap/social, reusable Section, StatBlock, ServiceCard, CaseStudyCard, CTAStrip.

### Assets
Use high-quality free stock imagery (Unsplash) for hero/section backgrounds; placeholder client logos and screenshots so nothing infringes on Holbi's actual customers.

### Out of scope (for now)
Blog, real backend, CMS, analytics, multi-language. Easy to add later.
